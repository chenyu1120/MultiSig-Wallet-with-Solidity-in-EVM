# MultiSig Wallet

This will show you how to build MultiSig Wallet in Solidity.

## Available Scripts

```shell
npm run compile
npm run test
npm run deploy-token-testnet
npm run deploy-multisigwallet-testnet
npm run deploy-token-mainnet
npm run deploy-multisigwallet-mainnet
```
